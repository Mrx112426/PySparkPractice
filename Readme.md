# Научно-исследовательская работа по PySpark

**Автор:** Мазориев У. А.

Репозиторий содержит научно-исследовательскую работу и практические материалы по использованию **PySpark** для распределённой обработки больших данных.

Основной фокус работы - исследование архитектуры Apache Spark, возможностей PySpark, различий между RDD и DataFrame API, а также практическое применение инструмента для аналитики разнородных наборов данных.

## Научно-исследовательская работа

- [НИР: Исследование инструмента PySpark для распределённой обработки больших данных](./НИР_PySpark_Мазориев.pdf)

В работе рассматриваются:

- место PySpark в современных системах обработки больших данных;
- архитектура Spark-приложения: driver, executors, cluster manager, DAG;
- модель RDD, механизм lineage и отказоустойчивость;
- DataFrame API, Spark SQL, Catalyst Optimizer и Tungsten;
- узкие и широкие преобразования, shuffle и партиционирование;
- практическая реализация обработки данных на RDD и DataFrame;
- сравнение PySpark с pandas, Dask, Apache Flink и Hadoop MapReduce;
- совместимость PySpark с форматами данных, хранилищами и инструментами data stack.

## Практические notebook

### RDD API

- [Практический notebook по RDD](./RDD/PySpark_rdd.ipynb)
- [README практики RDD](./RDD/README.md)

В ноутбуке разобраны задачи распределённой обработки на низкоуровневом RDD API:

- анализ логов промышленных IoT-сенсоров;
- анализ распространённых паролей;
- лингвистический анализ русского словаря;
- парсинг логов веб-сервера;
- анализ вовлечённости постов в социальной сети;
- анализ списка стран.

Ключевые операции: `map`, `filter`, `reduceByKey`, `sortByKey`, `distinct`, `count`, `mean`, `top`, `takeOrdered`.

### DataFrame API

- [Практический notebook по DataFrame](./DataFrame/PySparkDataFrame.ipynb)
- [README практики DataFrame](./DataFrame/Readme.md)

В ноутбуке разобраны задачи структурированной аналитики на PySpark DataFrame API:

- анализ активности онлайн-пользователей;
- анализ финансовых показателей ПАО «АвтоВАЗ»;
- анализ данных вселенной Star Wars.

Ключевые возможности: `SparkSession`, схемы данных, `MapType`, `ArrayType`, `withColumn`, `groupBy`, оконные функции `Window`, `join`, чтение CSV и Parquet.

## Цель проекта

Проект показывает, как PySpark может использоваться не только как инструмент выполнения отдельных аналитических задач, но и как основа для построения воспроизводимых конвейеров обработки больших данных. Практические notebook дополняют исследовательскую часть и демонстрируют применение теоретических механизмов Spark на конкретных задачах.
